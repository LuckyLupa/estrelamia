Passo a passo:
1. Custo da peca + R$ 17,36 = BASE
2. BASE + lucro desejado = TOTAL
3. Classico sem frete: TOTAL / 0,825
3. Classico com frete: TOTAL / 0,795
3. Premium com frete: TOTAL / 0,775

Exemplo: peca custa R$ 45, quer ganhar R$ 30
Base = 45 + 17,36 = 62,36
Total = 62,36 + 30 = 92,36
Preco = 92,36 / 0,825 = R$ 111,95
